map_1var

plots.maps.map_1var(
    da,
    *,
    levels,
    cmap='rain',
    shapefile=None,
    extent=None,
    xticks=None,
    yticks=None,
    tick_step=(5, 5),
    draw_grid=False,
    time=None,
    month=None,
    title=None,
    colorbar_label=None,
    output_path=None,
)

Draw a filled map of a single scalar variable.

The same temporal selection is applied to the data via: - month (for climatological datasets with a month coordinate), or - time (for datasets with a time coordinate; can be int index or str timestamp).

Parameters

Name Type Description Default
da xarray.DataArray Scalar field with coordinates (lat, lon) and optionally time or month. required
levels iterable of float Discrete color levels for the filled contours. required
cmap str Name of the cmocean colormap to use (lightened internally). 'rain'
shapefile str or None Path to a custom shapefile. If None, uses the default South America shape. None
extent tuple or None Map extent (lon_min, lon_max, lat_min, lat_max). If None, uses DEFAULT_EXTENT. None
time int or str or None If dataset has time, selects a single frame: index (int) or timestamp (str). None
month int or None If dataset has month, selects that month (1-12). None
title str or None Figure title. If None, uses long_name or variable name. None
colorbar_label str or None Colorbar label. If None, uses da.units if available. None
output_path str or None If provided, saves the figure to disk; otherwise shows the figure. None

Draw a filled map of a scalar variable and overlay wind vectors (U, V).

The same temporal selection (time or month) is applied uniformly to da, U, and V.

Parameters

Name Type Description Default
da xarray.DataArray Scalar field with (lat, lon) and optionally time/month. required
U xarray.DataArray Vector components with the same horizontal grid as da. required
V xarray.DataArray Vector components with the same horizontal grid as da. required
levels iterable of float Discrete color levels for the filled contours. required
cmap str Name of the cmocean colormap (lightened internally). 'rain'
shapefile str or None Custom shapefile path; None uses default SA shapefile. None
extent tuple or None Map extent (lon_min, lon_max, lat_min, lat_max). None
time see map_1var Temporal selector (applied to all inputs). None
month see map_1var Temporal selector (applied to all inputs). None
quiver_subsample int Arrow subsampling step to reduce density. 7
quiverkey_speed float Reference speed (m/s) shown in the quiver key. 5.0
title see map_1var. None
colorbar_label see map_1var. None
output_path see map_1var. None
