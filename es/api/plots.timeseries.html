Plot a time series (or monthly climatological cycle) averaged over a lat–lon box.

Notes

This function DOES NOT perform any temporal subsetting. It uses the full series provided by the user. Any prior selection (e.g., time ranges, month filtering, anomalies, climatologies) must be done upstream.

Parameters

Name Type Description Default
da xr.DataArray Input data array with ‘time’ or ‘month’ coordinate. required
lat_range tuple of float Latitude/longitude bounds (min, max) for area averaging. None
lon_range tuple of float Latitude/longitude bounds (min, max) for area averaging. None
resample str Resampling frequency (e.g., ‘M’, ‘Y’, ‘Q’). Only if ‘time’ exists. None
label optional Plot styling and output controls. 'Series'
color optional Plot styling and output controls. 'Series'
linestyle optional Plot styling and output controls. 'Series'
linewidth optional Plot styling and output controls. 'Series'
title optional Plot styling and output controls. 'Series'
ylabel optional Plot styling and output controls. 'Series'
output_path optional Plot styling and output controls. 'Series'

ts_point_1var

plots.timeseries.ts_point_1var(
    da,

    lat,
    lon=None,
    time=None,
    month=None,
    resample=None,
    labels=None,
    colors=None,
    linestyles=None,
    linewidth=1.8,
    title=None,
    ylabel=None,
    output_path=None,
)

Plot one or multiple point series (time or monthly) from a DataArray.

Notes

No temporal subsetting is performed here; the full input series is used. To subset in time or compute climatologies/anomalies, do it upstream.

Parameters

Name Type Description Default
da xr.DataArray Input data array with ‘time’ or ‘month’ coordinate. required
lat float | list | list[tuple] - Single point: lat=float and lon=float - Multiple points: [(lat1, lon1), (lat2, lon2), …] required
lon float | list | list[tuple] - Single point: lat=float and lon=float - Multiple points: [(lat1, lon1), (lat2, lon2), …] required
resample str Resampling frequency (e.g., ‘M’, ‘Y’, ‘Q’). Only if ‘time’ exists. None
labels list Styling per point series. Defaults will be generated if not provided. None
colors list Styling per point series. Defaults will be generated if not provided. None
linestyles list Styling per point series. Defaults will be generated if not provided. None
linewidth optional Plot styling and output controls. 1.8
title optional Plot styling and output controls. 1.8
ylabel optional Plot styling and output controls. 1.8
output_path optional Plot styling and output controls. 1.8

Deprecated Parameters

time, month Ignored. Kept for backward compatibility to avoid breaking old calls.
