    da,

    cmap='thermal',
    time=None,
    month=None,
    title=None,
    colorbar_label=None,
    output_path=None,
)

Plot an X–Z vertical section at a fixed latitude (lon vs vertical).

Parameters

Name Type Description Default
da xarray.DataArray Scalar variable with coordinates (time|month?, z, lat, lon). required
lat float Fixed latitude at which the section is extracted (nearest selection). required
levels iterable of float Discrete color levels for the filled section. required
cmap str cmocean colormap name (lightened internally). 'thermal'
time int or str Select a single time frame if available (index or timestamp string). None
month int Select a single month if available (1–12) for climatologies. None
title str Figure title. None
colorbar_label str Label for the colorbar; defaults to da.units if present. None
output_path str or None If provided, the figure is saved; otherwise it is shown. None

    title=None,

)

Plot an X–Z section (fixed latitude) of a scalar field and overlay wind vectors (U, W).

Parameters

Name Type Description Default
da xarray.DataArray Scalar field (time|month?, z, lat, lon). required
U xarray.DataArray Zonal wind component on the same grid (time|month?, z, lat, lon). required
W xarray.DataArray Vertical wind component on the same grid; will be scaled by w_scale. required
lat float Fixed latitude for the section (nearest selection). required
levels iterable of float Discrete color levels for the filled section. required
cmap str cmocean colormap name (lightened internally). 'thermal'
time optional Uniform temporal selector applied to all inputs. None
month optional Uniform temporal selector applied to all inputs. None
w_scale float Multiplicative factor for W (e.g., 1000 to convert m/s to mm/s for better visual balance). 1000.0
quiver_density int Subsampling step for the quiver field (larger -> fewer arrows). 4
quiver_scale float Matplotlib quiver scaling parameter. 170.0
quiverkey_speed float Reference speed (m/s) shown in the quiver key. 5.0
title Optional[str] See section_xz_1var. None
colorbar_label Optional[str] See section_xz_1var. None
output_path Optional[str] See section_xz_1var. None

Returns

Name Type Description
None
