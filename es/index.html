¿Qué es SCAHpy?

SCAHpy (System for Coupled Atmosphere–Hydrosphere Analysis in Python) es un paquete de Python de código abierto diseñado para facilitar el análisis, diagnóstico y visualización del Modelo del Sistema Terrestre Regional del IGP (IGP-RESM-COW), un sistema acoplado océano-atmósfera que cubre el territorio peruano y el océano Pacífico oriental.

El modelo IGP RESM-COW integra: el modelo atmosférico WRF, el modelo oceánico CROCO y, el acoplador OASIS.

Este sistema numérico ha sido implementado y operado por el Instituto Geofísico del Perú (IGP) para estudios de dinámica oceánica, procesos atmosféricos y variabilidad climática en la región peruana.

Documentación técnica del sistema acoplado (IGP RESM-COW):

  • Modelado atmosférico regional y configuración WRF
  • Configuración oceánica regional con CROCO
  • Implementación y validación del sistema acoplado
  • Aplicaciones del modelo en estudios del clima costero peruano

¿Por qué SCAHpy?

Tanto la componente oceánica como atmosférica del modelo IGP RESM-COW genera grandes volúmenes de información NetCDF, usualmente distribuidos en múltiples archivos diarios, mensuales o multianuales.
Procesar estos datos puede ser retador debido a diferencias en:

  • dimensiones y convenciones de coordenadas,
  • niveles verticales (sigma, niveles de presión),
  • calendarios y zonas horarias,
  • concatenación temporal,
  • extracción de cortes horizontales y verticales,
  • generación de productos reproducibles para análisis científico.

SCAHpy fue creado para resolver estos desafíos proporcionando:

  • funciones estandarizadas para leer salidas WRF y CROCO,
  • manejo robusto de coordenadas y dimensiones,
  • transformación vertical (sigma → presión),
  • diagnósticos atmosféricos y oceánicos,
  • visualización de alto nivel (mapas, secciones, series de tiempo),
  • flujos de trabajo reproducibles tanto en entornos locales como en HPC.

SCAHpy reduce significativamente el tiempo necesario para el posprocesamiento científico y permite enfocarse en la interpretación física de los resultados.

¿Cómo usar SCAHpy?

Para comenzar, consulta la sección Uso y los tutoriales presentados más abajo.

Nota

SCAHpy ha sido desarrollado y validado principalmente con salidas del sistema IGP RESM-COW; sin embargo, es compatible con salidas individuales del modelo WRF, CROCO o dataset NetCDF que siga las CF-Conventions.
